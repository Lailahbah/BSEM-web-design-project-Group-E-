# SaLone Market 🇸🇱

**Sierra Leone's Digital Market** — connecting farmers, traders, and buyers on one platform.

SaLone Market is a frontend web platform built to help local farmers and traders sell their goods directly to buyers across Sierra Leone, while giving buyers reliable access to fresh, organic produce at fair prices.

## Features

- **Responsive navigation** with a mobile hamburger menu
- **Hero section** showcasing fresh local produce
- **About Us** section explaining the platform's mission
- **Why Choose Us** highlights (fresh & organic goods, ease of use, up-to-date listings, nationwide access)
- **Key Categories** browsing (Grains & Rice, Fresh Fruits, Root Tubers, Fresh Fish)
- **News & Updates** feed for market announcements
- **Testimonials** from farmers, traders, and buyers
- **Platform stats** (active users, farmers, districts covered, satisfaction rate)
- **Newsletter signup**
- **Shopping cart** sidebar with item count and toast notifications
- **Footer** with platform/support/legal links and SDG alignment badges

## 🛠️ Tech Stack

- CSS3 (`css/style.css`)
- Vanilla JavaScript (`js/script.js`)

## 🚀 Getting Started

1. Clone or download this repository.
2. Make sure the `css/`, `js/`, and `images/` folders are in the same directory as the HTML files.
3. Open `index.html` in your browser, or serve the folder with a simple local server, e.g.:
   ```bash
   npx serve .
   ```
4. Navigate the site via the navbar: Home, Features, Browse Product, How It Works, and Join Free.
